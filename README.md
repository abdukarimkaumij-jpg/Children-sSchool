Children School — это адаптивный веб-сайт для детского образовательного центра, разработанный с акцентом на удобство пользователя и современный дизайн.

В рамках проекта был реализован полноценный интерфейс сайта, включающий:

главную страницу с презентацией школы
блоки с направлениями обучения
галерею с фильтрацией контента
информационные секции (о школе, преподавателях)
форму записи на занятия
⚙️ Используемые технологии:
HTML5
CSS3 (SCSS)
JavaScript
Адаптивная верстка (Responsive Design)
Flexbox / Grid
🚀 Реализованный функционал:
фильтрация элементов по категориям
интерактивные UI-компоненты
адаптация под мобильные устройства
структурированная и семантическая верстка
🎯 Цель проекта:

Создание современного и удобного сайта для привлечения клиентов и презентации образовательных услуг детского центра.

🧑‍💻 Моя роль:

Frontend-разработчик — разработка структуры, верстка интерфейса и реализация интерактивности.
